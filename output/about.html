About

Erdos has been inspired by projects like Project Euler for math and programming, ROSALIND for bioinformatics, and the cryptopals challenges for cryptography.

Erdos is open source and free of charge. You can contribute directly on our GitHub page, here. Erdos uses Python to generate the site, with the help of Pelican, a modification of elegant-theme that can be found here, and markdown-pp. Most solutions are also written in Python.

If you have any questions, comments, suggestions, or would like to contribute to Erdos, we would like to hear from you! You can find the contact information to the right of this page.
